## 🧠 Project Overview
BidX is a frontend-driven auction system featuring role-based interaction:
- **Admin** can add and manage auction items  
- **Bidders** can place bids and compete in real time  

The system tracks the highest bid and simulates auction logic with an intuitive interface.
